Integrated Database & Macroeconomic Modelling

Project by Admire T. Chirume

Project Overview

This project develops an integrated macroeconomic database linked to a simulation-based forecasting and modelling system. Built with SQLite and R (via R Markdown), it supports automated data processing, forecasting, and scenario analysis for policy use.

Technical Architecture

  • Database: SQLite (local file-based)
  • Modeling Language: R (tidyverse, forecast, vars, ML packages)
  • Forecasting Models: VAR, ARIMA, ML (XGBoost, LSTM via Keras in R)
  • Interface: R Markdown → HTML / PDF (hosted via GitHub Pages & RPubs)
